->Read the complete plan.md file first and fully understand the entire project before starting.
->Review all tasks and steps defined in plan.md.
->Before executing any step, identify the required skills.
->Go to the .agents/skills/ directory.
->First, use the find-skills skill to learn how to discover and select the appropriate skills.
->Based on the guidance from find-skills, choose:
->Global Skills – skills that will be used throughout the entire project.
->Step-Specific Skills – skills required only for an individual step.
->Assign the appropriate skills to every step in plan.md.
->Create a new document named AgentSkills.md.
->In AgentSkills.md, include:
->A Global Skills section listing all skills used across the project.
->A Step-by-Step Skills section mapping each step in plan.md to its required skills.
->Ensure every step has the correct skills assigned before execution begins.
->Do not execute any tasks until the skill assignment is complete.
