# find-skills

A metadata skill for routing, finding, and loading specialized instruction sets among the 1,440+ available agent skills.

## Quick Start

### 1. Find a Skill via CLI Router
To query all skills:
```bash
python3 find-skills/scripts/router.py "django query optimization"
```

### 2. Browse Category Reference
Open [references/all_skills_detailed.md](references/all_skills_detailed.md) to browse skills sorted by category.
