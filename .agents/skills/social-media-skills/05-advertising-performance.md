Create, iterate, and scale paid ad creative for Meta (Facebook/Instagram), Google Ads, LinkedIn, TikTok.

### Meta Ads Specs (Facebook/Instagram)
| Element | Limit | Notes |
|---------|-------|-------|
| Primary text | 125 chars visible (up to 2,200) | Front-load the hook |
| Headline | 40 characters recommended | Below the image |
| Description | 30 characters recommended | Below headline |
| URL display link | 40 characters | Optional |

### How It Works

**Mode 1: Generate from Scratch**
1. Define 3-5 distinct angles (pain point, outcome, social proof, curiosity, comparison, urgency)
2. Generate variations per angle (vary word choice, specificity, tone, structure)
3. Validate against character limits
4. Organize for platform upload

**Mode 2: Iterate from Performance Data**
1. Analyze winners (themes, structures, word patterns)
2. Analyze losers (angles that fall flat)
3. Generate new variations (double down on winning themes, test new angles)
4. Document iteration log

### Angle Categories
| Category | Example |
|----------|---------|
| Pain point | "Stop wasting time on X" |
| Outcome | "Achieve Y in Z days" |
| Social proof | "Join 10,000+ teams who..." |
| Curiosity | "The X secret top companies use" |
| Urgency | "Limited time: get X free" |
| Identity | "Built for [specific role]" |

### Batch Generation Workflow
- **Wave 1:** Core angles (3-5 angles, 5 variations each)
- **Wave 2:** Extended variations on top 2 angles
- **Wave 3:** Wild card angles (contrarian, emotional, specific)

## marketing-psychology
**Path:** `/Users/charankumar/skills/marketing-psychology/`
**Source:** community

Apply behavioral science and mental models to marketing decisions using Psychological Leverage & Feasibility Score (PLFS).

### PLFS Formula
```
PLFS = (Leverage + Fit + Speed + Ethics) - Implementation Cost
```
Range: -5 to +15. Score 12-15 = Apply immediately, 8-11 = Prioritize, 4-7 = Test, 1-3 = Defer, ≤0 = Do not recommend.

### Journey-Based Model Selection
| Stage | Key Models |
|-------|-----------|
| Awareness | Mere Exposure, Availability Heuristic, Authority Bias, Social Proof |
| Consideration | Framing Effect, Anchoring, Jobs to Be Done |
| Decision | Loss Aversion, Paradox of Choice, Default Effect, Risk Reversal |
| Retention | Endowment Effect, IKEA Effect, Status-Quo Bias |

### Mandatory Rules
- Never recommend more than 5 models
- Each must map to a specific behavior
- Each must include an ethical note
- Never recommend models with PLFS ≤ 0

### Ethical Guardrails
- No dark patterns, false scarcity, hidden defaults
- Must ensure transparency, reversibility, informed choice

## seo-content-writer
**Path:** `/Users/charankumar/skills/seo-content-writer/`
**Source:** community

Write SEO-optimized content for blog posts, articles, and web pages.

### Content Creation Framework
- **Introduction (50-100 words):** Hook + value proposition + primary keyword
- **Body:** Logical flow, supporting data, natural keyword placement, clear subheadings (H2/H3)
- **Conclusion:** Summary + CTA

### Quality Standards
- 0.5-1.5% keyword density
- Grade 8-10 reading level
- Short paragraphs (2-3 sentences)
- Bullet points for scannability

### E-E-A-T Elements
- First-hand experience mentions
- Specific examples and cases
- Data and statistics citations
- Expert perspective

### Output Package
- Full article at target word count
- 3-5 title variations
- Meta description (150-160 chars)
- Key takeaways
- Internal linking suggestions
- FAQ section if applicable
