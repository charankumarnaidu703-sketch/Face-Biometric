Expert social media strategist with platform-specific playbooks for all major networks.

### Platform Guides Included

#### Instagram Strategy
- **Best for:** Visual brands, lifestyle, e-commerce, younger demographics (18-44)
- **Posting frequency:** 1-2 feed posts/day, 3-10 Stories/day
- **Best times:** 11am-1pm, 7-9pm
- **What works:** High-quality visuals, Reels (2x reach of static), carousels, UGC, interactive Stories
- **Format tips:** First frame of Reels must hook, carousels: 10 slides educational, use all Story features

#### Facebook Strategy
- **Best for:** Communities, local businesses, older demographics (25-55+), groups
- **Posting frequency:** 1-2x per day
- **Best times:** 1-4pm weekdays
- **What works:** Facebook Groups, native video, live video, local content, discussion prompts
- **Avoid:** External links (reach killer), pure promotion, cross-posting without adaptation

### Content Pillars Framework
Build 3-5 content pillars. Example mix: Industry insights (30%), Behind-scenes (25%), Educational (25%), Personal (15%), Promotional (5%)

### Instagram Caption Templates
- **Carousel Hook:** Slide 1 = bold statement, Slides 2-9 = one point each, Slide 10 = summary + CTA
- **Reel Script:** Hook (0-2s) -> Setup (2-5s) -> Value (5-25s) -> CTA (25-30s)

### Hook Formulas
| Type | Examples |
|------|----------|
| Curiosity | "I was wrong about [common belief]." |
| Story | "Last week, [unexpected thing] happened." |
| Value | "How to [outcome] (without [common pain]):" |
| Contrarian | "Unpopular opinion: [bold statement]" |
| Social Proof | "We [achieved result] in [timeframe]. Here's how:" |

### Content Repurposing System
Blog post -> LinkedIn carousel / Twitter thread / Instagram carousel / Reel
Podcast/video -> Quote graphic / Clip as Reel / YouTube Short

### Analytics Tracking
- **Awareness:** Impressions, reach, follower growth rate
- **Engagement:** Engagement rate, comments, shares, saves
- **Conversion:** Link clicks, profile visits, DMs, leads
